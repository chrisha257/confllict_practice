1. 아래 명령어를 통해 클론을 받는다.
```bash
git clone URL
```
2. 로컬에서 commit을 하나 더 쌓은 다음에 remote로 push한다.
3. 리모트에서 commit을 하나 더쌓은 다음 loacl로 pull한다.
4. 로컬에서 commit을 쌓습니다.
  - 로컬에서 commit을 쌓습니다.
  - 리모트에서 commit을 쌓습니다.
5. push를 하려고 했더니 regected가 뜹니다.
6. 이를 해결하기 위해 pull을 먼저합니다.
7. merge commit을 남깁니다.
